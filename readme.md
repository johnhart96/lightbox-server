# Lightbox-server
Lightbox-server is a network manager for entertainment networks. It provides:

* DHCP
* DNS
* NTP
* Samba file sharing

These services can be configured using a simple web interface.

It is designed to run on a raspberry pi, but in theory will run on any debian or ubuntu device.

## Key features
* Supports both IPv4 and IPv6
* Automatically adds DHCP leases to DNS
* Allows for custom DNS entries to be added
* View all service logs in the web interface
* Configure network interfaces via the web interface
* Monitor server resource usage via web interface.
